# 🚀 AdFlow Pro

AdFlow Pro is a full-stack ads management web application built using React, Node.js, Express, and MongoDB.

---

## 🌟 Features

- Create Ads (Title, Description, Image, Category)
- Admin Panel (Approve / Reject Ads)
- Search Ads 🔍
- Filter by Categories 📂
- Modern UI (Glassmorphism Design)

## 🧰 Tech Stack

- Frontend: React.js  
- Backend: Node.js + Express.js  
- Database: MongoDB  
- API Calls: Axios

## 🔌 API Routes

- GET / → Test API  
- POST /ads → Create Ad  
- GET /ads → Approved Ads  
- GET /admin/ads → All Ads  
- PUT /ads/:id → Update Status
